## 🚀 运行说明

### 1. 安装依赖
```bash
cd mobile
npm install
```

### 2. 运行到模拟器
```bash
# Android
ns run android

# iOS
ns run ios
```

### 3. API 配置
已配置开发环境访问：
- Android: `http://10.0.2.2:8000/api/v2`
- iOS: `http://localhost:8000/api/v2`
